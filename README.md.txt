# MySQL E-commerce Project

## Description
A simple database to manage products, customers, orders and payments.

## Technologies
- MySQL (XAMPP)
- phpMyAdmin

## Features
- Stored Procedure to place an order
- Trigger to prevent overselling
- Function to calculate customer total

## How to Run
1. Create database `ecommerce`
2. Import `schema.sql`
3. Import `data.sql`
4. Add `procedures.sql`, `functions.sql`, and `triggers.sql`
5. Run `queries.sql` to test
